# go-http-authorizer
HTTP Authorizer API using Casbin library

## Endpoints:
"/check" - check authorize request with current policy

"/policies/add" - POST: create new policy

"/policies/delete" - DELETE: remove policy

"/policies/get" - GET: get all policies
